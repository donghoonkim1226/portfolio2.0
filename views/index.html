Clicky Game

Using HTML, CSS, and JavaScript designd a simple app that has a grid of 20 images. By using basic javascript the player is able to press the "Start" button that kicks off a 20 second timer and then count the number of faces a player can click in that time. Once the 20 seconds is up, the user is alerted with how many faces they have clicked. Multiple clicks of the same image does not work.
